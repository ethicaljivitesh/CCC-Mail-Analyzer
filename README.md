# CCC Mail Forensic Analyzer
This is a Python-based application for analyzing email data and extracting information such as headers, body content, and IP addresses.

## Features:
- Upload and analyze `.eml` email files.
- Trace IP addresses found in email headers.
- Generate an HTML report for email analysis.
- Keyword search functionality within the email body.

## Requirements:
- Python 3.x
- Install dependencies using the `requirements.txt` file.
